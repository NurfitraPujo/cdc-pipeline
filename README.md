# Daya Data Pipeline
This repository is used to build data pipelines for daya. This repo is monorepo consisted of FE and BE services. FE service is to help monitor the data pipeline while BE service is to process the data.

# Prerequisite
- Go 1.26.x or higher
- Docker / Docker compose or equivalent container runtime
- Postgresql 15 or higher
- NATS (default streaming server)
