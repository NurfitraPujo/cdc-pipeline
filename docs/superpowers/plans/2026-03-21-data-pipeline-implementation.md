# Daya Data Pipeline Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement a modular CDC data pipeline from PostgreSQL to Databend via NATS.

**Architecture:** Producer-Consumer decoupled by NATS JetStream (via Watermill), with 2-step tracking in NATS KV.

**Tech Stack:** Go 1.26+, NATS, Watermill, go-pq-cdc, Databend, TanStack Start.
